## Intro
![bg left:40%](../img/robot7.jpg)

# Overview
> Conditions are executed once at most, loops possibly multiple times.

# Conditions
```python
if sunUp:
    print ("Hallo Sonne!")
if onEarth:
    print ("Hello World")
else:
    print ("Hello Aliens")
```
![](../img/scratch/conditions/conditions.svg)
# Loops
```python
for x in range(10):
    print (x)
while (sunUp):
    print ("A day without sunshine, is, you know, night.)
```
