# Variables
![bg left:40%](../img/robot6.jpg)

## What are variables

* Places / Names to store values
* Can be Ordinals, Text, List, Objects
* Name is a textual reference
* Calulation is possible

## Ordinals

```python
c = 10
pi = 3.1415
sunUp = True
radius = 10
circumfence = 2*radius*pi
```
